với yêu cầu 8:
- chuyển sang branch được yêu cầu nếu chưa ở đó:
+ kiểm tra branch hiện tại đang ở : git branch
+ git checkout “tên branch”
- tạo branch mới và chuyển sang branch này 
+ git checkout -b “tên branch”

với yêu cầu 14 merge 2 branch:
- chuyển sang branch cần merge : git checkout feature-truongvd
- thực hiện merge : git merge feature-truongvd-merge
- sau đó có một cửa sổ hiện các conflict
- nếu có conflict thì sẽ mở file và xử lý
